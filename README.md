# 🟢 Faro's Dynamic Portfolio & CMS

> A high-performance, fully dynamic personal portfolio and Content Management System (CMS), built to showcase my journey and projects in Software Engineering, Artificial Intelligence, Automation, and Hardware.

## 📖 About The Project
This platform is more than just a static resume; it's a complete web application built from scratch using **Python (Flask)**. Designed with a sleek, dark Cyberpunk/Matrix-inspired aesthetic (featuring Neon Green accents and Glassmorphism), it serves as a centralized, interactive hub for all my technical endeavors.

The platform includes a secured, built-in **Admin Dashboard** that allows for seamless content updates, project management, and message handling—meaning the portfolio scales and grows dynamically as I do.

## ✨ Key Features
* **Dynamic CMS (Admin Dashboard):** Securely add, edit, and delete projects, update profile settings, manage avatars, and read incoming contact messages.
* **Markdown Support:** Project details and case studies are written in Markdown and rendered dynamically into beautifully styled HTML using `Marked.js`.
* **Advanced UI/UX Elements:** * Cyberpunk/Terminal visual theme with frosted glass effects.
  * Dynamic "Typing Effect" terminal cursor.
  * Scroll Reveal animations for smooth element loading.
  * Custom neon scrollbars and interactive hover glow effects.
  * **Fully Responsive:** Flawless experience across desktop, tablet, and mobile screens.
* **Categorized Showcases:** Projects are efficiently organized into three main sectors: `Software & AI`, `Automation`, and `Arduino & Hardware`.
* **Database Integration:** Utilizes a lightweight and fast `SQLite` database to handle all dynamic content and media paths.

## 🛠️ Tech Stack
* **Backend:** Python, Flask, SQLite, Werkzeug 
* **Frontend:** HTML5, CSS3 (Custom Animations & Glassmorphism), Vanilla JavaScript, Marked.js
* **UI Details:** FontAwesome (Icons), Fira Code & Quicksand (Typography)
